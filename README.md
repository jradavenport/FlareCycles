# FlareCycles
Searching for activity cycles using variations in stellar flare rates. This should be Paper 1: Initial searches of a few high quality stars.

Supported by TESS Cycle-3 [GO3227](https://heasarc.gsfc.nasa.gov/docs/tess/data/approved-programs/cycle3/G03227.txt)

Subject of a few funding proposals (fingers crossed)

Idea outlined in [Scoggins+2019](https://ui.adsabs.harvard.edu/abs/2019RNAAS...3..137S)
